Itera — Design System

Este archivo es el espejo fiel del design system actual de Itera (simulador corporativo B2B que mide criterio en el uso de IA). Generado 1:1 desde el código fuente: los tokens vienen de simulador.css y cada componente está recreado leyendo su implementación real. Cada demo aparece en par claro / oscuro.

Cómo usarlo: trabaja y corrige SOBRE estas piezas — evoluciona tokens y componentes, no inventes una librería nueva. Todo cambio que propongas debe poder expresarse como valores de estos tokens (hex/px/ms) porque se aterrizan a variables CSS que propagan a toda la aplicación.
