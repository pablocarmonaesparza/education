Pablo y Codex redefinieron la hipotesis de producto de Itera tras revisar el problema de fondo: el curso se sentia demasiado teorico y el beneficio para empresas era dificil de tangibilizar.

**Direccion nueva:** Itera no debe posicionarse como "otro curso de IA", ni como herramienta de automatizacion tipo n8n/Make/Zapier, ni como wrapper de ChatGPT/Claude/Gemini. El espacio mas defendible es:

> Itera es un simulador empresarial para practicar criterio IA en situaciones reales de trabajo.

**Lo que significa:**
- La unidad principal deja de ser solo "leccion" y empieza a acercarse a "simulacion".
- La evidencia para empresa no tiene que ser ROI duro inmediato ni workflow automatizado en produccion.
- La evidencia debe mostrar readiness: criterio, calidad de instrucciones, evaluacion de respuestas, manejo de riesgo, privacidad, decision de herramienta y capacidad de aplicar IA con supervisacion razonable.
- Itera sigue siendo educacion. No ejecuta el trabajo por la empresa; entrena al empleado para usar mejor las herramientas existentes.

**No competir contra:**
- ChatGPT, Claude, Gemini, Codex, Veo, Nano Banana u otros motores generativos.
- n8n, Zapier, Make, CRMs, chatbots o plataformas de automatizacion.
- LMS genericos cuyo valor principal es catalogo, video, rutas y certificados.

**Interseccion buscada:**

```
herramientas IA -> situacion laboral -> practica guiada -> criterio -> evidencia para empresa
```

**Comparables a investigar antes de implementar:**
- Wharton Interactive: simulaciones de negocio para aplicar criterio en entornos seguros.
- Forage: job simulations con tareas que replican trabajo real.
- Attensi / Mursion / Whatfix Mirror: simulacion, roleplay, readiness y performance medible.
- Section AI / DataCamp / Coursera / Udemy Business: adopcion y upskilling IA para empresas.
- Platzi Business / Crehana / Coderhouse: training corporativo LATAM y brecha local.

**Por que importa:** el mercado ya tiene acceso a IA. El problema de las empresas no es acceso ni informacion; es desarrollar criterio distribuido para usar IA de forma segura, productiva y medible en el trabajo.

**Estado:** direccion estrategica acordada para investigacion. No implica construir features todavia. Antes de pivotear producto/copy/landing/onboarding, hacer benchmark de competencia y categoria.

**Frase operativa:** "Itera es un flight simulator para que equipos practiquen decisiones reales con IA antes de aplicarlas en el trabajo real."
